# Gym Trainer Backend

This backend is designed for the Gym Trainer Webapp. It uses Node.js, Express, and MySQL to store and manage user logins, user info, training info, subscription info, and payment info.

## Features
- User authentication and management
- Training program management
- Subscription management
- Payment records

## How to Run
1. Install dependencies: `npm install`
2. Create a `.env` file with your MySQL credentials (see `.env.example`)
3. Start the server: `npm start`

## Database Schema
- See `schema.sql` for the MySQL database structure.
